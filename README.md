### CYPRESS-HILL

Proyecto de test E2E con Cypress sobre proyecto frontend React.

Instalación:
  - npm install

Run cypress
  - npx cypress open
#### Proyecto 1: Gestión de hábitos y logros (frontend)

Archivos de tests:  
cypress / integration
  - accomplishment.spec.js  
  - habits.spec.js  
  - locators.spec.js  

Métodos
  - Get
  - Contains
  - Find
  - Should/And
  - Type
  - Click

Comandos
  - Add/Get
